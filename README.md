# planner — KROLIK edition 🐰

Персональный планировщик: быстрый захват мыслей, задачи, заметки-граф с иерархией, календарь, доска, корзина и таймер фокуса.
Нативное окно (pywebview / WebView2), всё локально, без интернета.

## Запуск
Двойной клик по **`planner.bat`** (или ярлык на рабочем столе).

## Где данные
- `data/planner.json` — все заметки/задачи/области (автосохранение).
- `data/backups/` — авто-бэкапы (снимок при каждом старте + кнопка «бэкап»), хранятся последние 30.
- Экспорт/импорт в любой файл — кнопки слева внизу.

## Горячие клавиши
- `Ctrl+K` — поиск / командная палитра
- `/` — фокус в поле захвата
- `n` — новый элемент
- `1…7` — переключение видов (Сегодня / Inbox / Задачи / Заметки / Календарь / Доска / Корзина)
- `Esc` — закрыть окно/режим связи

## Быстрый захват (поле сверху)
Пишешь мысль, `Enter`. В строке можно указать:
- `#область` — например `#работа`, `#прочее`, `#аддоны` (по началу названия)
- даты: `сегодня`, `завтра`, `послезавтра`, `пн…вс`, `25.06`, `через 3 дня`, `через 2 недели`
- повтор: `ежедневно`, `еженедельно`, `ежемесячно`
- приоритет: `!` (можно `!!`, `!!!`)

Без даты мысль падает в **Inbox**. С датой — сразу в задачи.

## Заметки (граф + иерархия)
- Узлы: области (○ залитый), заметки (○ контур), задачи (▢).
- Линии: тонкие — связь с областью; яркие со свечением — твои ручные связи.
- Тяни узлы мышкой, колесо — масштаб, фон тянется панорамой.
- Двойной клик по узлу — открыть заметку/задачу или перейти к области.
- Клик по узлу → меню: изменить / связать / удалить. «Связать» → клик по второму узлу.
- **Иерархия автоматическая**: когда ты связываешь две заметки на графе (ПКМ → «Связать» → клик по второму узлу), вторая заметка автоматически становится дочерней первой. В списке заметок она отображается под родительской с отступом. Удаление связи = разрыв иерархии.
- На графе дочерние узлы физически тянутся к родителю (сильная пружина).

## Задачи
- Приоритет: `!` — низкий, `!!` — средний, `!!!` — высокий (цветная полоса слева).
- Сортировка: срочные всегда наверху.
- Повторы: при выполнении повторяющейся задачи создаётся новая копия (сохраняется описание).

## Доска (Kanban)
- 4 колонки: Inbox / Запланировано / В работе / Готово.
- Перетаскивание карточек между колонками (drag & drop).
- Двойной клик по заголовку колонки — переименовать.

## Корзина
- Удаление = soft-delete (в Корзину).
- Можно восстановить или удалить навсегда.
- Очистить всё — кнопка в заголовке Корзины.

## Таймер фокуса (Pomodoro)
- Кнопка часов в футере сайдбара.
- 25 минут фокуса — старт, пауза, сброс.

## Прочее
- **Прогресс по областям**: в сайдбаре рядом с каждой областью показывается % выполненных задач.
- **Напоминание при старте**: если есть просроченные задачи — toast-уведомление.
- **Цвета**: для областей и заметок можно выбирать цвет (в меню узла или редактора области).
- **Темы**: тёмная / светлая — кнопка солнца/луны в футере.

## Структура
```
planner/
  app.py            — окно + сохранение (Python)
  planner.bat       — запуск
  ui/               — интерфейс (index.html, styles.css, app.js)
  ui/vendor/        — иконки Tabler (локально)
  data/             — данные и бэкапы (создаётся при старте)
  .venv/            — окружение Python с pywebview
```

## Переустановка окружения (если понадобится)
```
python -m venv .venv
.venv\Scripts\python.exe -m pip install pywebview
```
