# Мыслик

Личное пространство для мыслей: задачи, заметки и схемы на одном холсте-паутине.
Десктопное приложение для Windows, работает офлайн, все данные — локально рядом с программой.

<img src="ui/icon.png" width="72" alt="Мыслик">

## Возможности

- **Быстрый захват** — брось мысль сверху и Enter. Понимает `#область/#тег`, даты (`завтра/пн/25.06`), `каждый день`, `!приоритет`, `*заметка`.
- **Сегодня** — кольцо дня, серия, ритм недели, баланс областей.
- **Заметки** — граф-паутина: наведи на ноду — подсветятся соседи; колесо — зум; фон-звёзды с параллаксом. Есть режим «Список».
- **Полотно** — блок-схемы на холсте (в духе Figma): блоки, стрелки, картинки/видео перетаскиванием, выравнивание по направляющим.
- **Папки** — привязка папок проектов на ПК, открытие в проводнике одним кликом.
- **Задачи** — иерархия с фильтром; инбокс раскидывается по областям кликом.
- **Календарь** — перетаскивание задач по дням, призраки повторов.
- **Telegram** — захват заметок с телефона через своего бота.
- **Обновление из приложения** — Настройки → Обновления → «Проверить обновления».

## Установка (для пользователя)

1. Скачай архив из [релизов](https://github.com/Krolik5555/myslik/releases/latest), распакуй папку `Myslik` целиком.
2. Запусти `Myslik.exe`. Windows может показать «защита компьютера» — «Подробнее» → «Выполнить в любом случае».
3. Данные хранятся в `data\` рядом с exe. Чтобы перенести — копируй всю папку.

Требуется Windows 10/11 x64 + Microsoft Edge WebView2 Runtime (обычно уже есть).

## Стек

Python + [pywebview](https://pywebview.flowrl.com/) (WebView2) — нативное безрамочное окно.
Интерфейс — vanilla HTML/CSS/JS без сборки (`ui/`), модули грузятся классическими `<script>`.
Данные — `data/planner.json` + авто-бэкапы в `data/backups/`.

## Сборка из исходников

```powershell
# зависимости
py -3.12 -m venv .venv
.\.venv\Scripts\pip install pywebview pystray pillow pywin32 pyinstaller

# запуск в дев-режиме
.\.venv\Scripts\python app.py

# портативный exe -> release\Myslik\
.\.venv\Scripts\python -m PyInstaller --noconfirm --distpath release --workpath build Myslik.spec
```
