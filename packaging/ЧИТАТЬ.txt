Мыслик 1.4.4.2 beta
====================================
Личное пространство для мыслей: задачи, заметки и схемы на одном холсте-паутине.

КАК ЗАПУСТИТЬ
--------------------
1. Распакуйте папку "Myslik" целиком в любое удобное место.
2. Запустите Myslik.exe (двойной клик).
   * Windows может показать синее окно "Система Windows защитила ваш компьютер".
     Нажмите "Подробнее", затем "Выполнить в любом случае".
3. Готово.

ГДЕ ХРАНЯТСЯ ВАШИ ДАННЫЕ
--------------------
* Всё лежит в папке  data\  рядом с Myslik.exe (data\planner.json + data\backups\).
* Чтобы перенести/забэкапить — скопируйте ВСЮ папку "Myslik" вместе с data.
* НЕ удаляйте папку  _internal  — без неё программа не запустится.

ОБНОВЛЕНИЕ
--------------------
* Настройки (шестерёнка внизу слева) -> раздел "Обновления" -> "Проверить обновления".
* Если вышла новая версия — кнопка "Обновить": приложение само скачает её, обновится
  и перезапустится. Ваши данные (папка data) при этом не трогаются.

С ЧЕГО НАЧАТЬ
--------------------
* Сверху — быстрый захват: бросьте мысль и Enter. Понимает #область/#тег, даты
  (завтра/пн/25.06), каждый день, !приоритет, *заметка.
* "Сегодня" — кольцо дня, серия, ритм недели, баланс областей.
* "Заметки" — граф-паутина: наведи на ноду — подсветятся соседи; ПКМ — создать;
  Alt+тащить от ноды — связь; колесо — зум; лупа — поиск. Есть режим "Список".
* Двойной клик по ноде-схеме — "Полотно": блоки кнопками сверху, тащи блок —
  двигать, 2x клик — переименовать, ПКМ — тип/цвет; картинки/видео перетаскиванием.
* "Папки" — привязка папок проектов на ПК, открытие в проводнике одним кликом.
* "Задачи" — иерархия с фильтром; инбокс раскидывается по областям кликом.
* "Календарь" — перетаскивание задач по дням; призраки повторов.
* Ctrl+K — палитра команд и поиск.

ЭКСПОРТ / ИМПОРТ / TELEGRAM
--------------------
* Внизу сайдбара: стрелка вниз — сохранить в файл, стрелка вверх — загрузить.
* Бэкап и тема — в Настройках.
* Захват мыслей с телефона: заведи бота у @BotFather, вставь токен в Настройки ->
  Telegram, пиши боту, в Inbox жми "Telegram" — сообщения прилетят.

ТРЕБОВАНИЯ
--------------------
* Windows 10/11 (64-бит) + Microsoft Edge WebView2 Runtime (обычно уже есть;
  иначе бесплатно: https://developer.microsoft.com/microsoft-edge/webview2/ , x64).
